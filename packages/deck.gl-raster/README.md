# @s3-cog/deck.gl-raster

> Derived from [Development Seed's deck.gl-raster](https://github.com/developmentseed/deck.gl-raster) monorepo (MIT), vendored into this repo — see [LICENSE](./LICENSE). Not this project's original work.

Core primitives for rendering georeferenced image data.

This image data can come from _any_ source. You only need to provide functions for associating pixel positions with spatial positions.

Users with Cloud-Optimized GeoTIFF data should use the higher-level {@link COGLayer} from the `@s3-cog/deck.gl-geotiff` package instead.

## Install

```bash
npm install @s3-cog/deck.gl-raster
```
